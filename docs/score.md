# 🎯 バレンタイン戦略 — 成功判定・スコアリング定義書

## 1. 成功の定義

### 1.1 基本方針

「お返しの有無」だけでは成功を正しく測れない。バレンタインの成功は、金銭的リターン・関係性リターン・感情的リターンの3層で評価する。

### 1.2 成功の3層モデル

| 層      | 名称           | 説明                             | 測定方法                             |
| ------- | -------------- | -------------------------------- | ------------------------------------ |
| Layer 1 | 金銭的リターン | ホワイトデーのお返し有無・金額   | 投資額に対する回収倍率（定量）       |
| Layer 2 | 関係性リターン | 渡したことで関係性が前進したか   | ユーザーの主観評価 + LLM推定（定性） |
| Layer 3 | 感情的リターン | 渡すこと自体の満足感・自己目的性 | ユーザーの気持ち入力（定性）         |

### 1.3 成功タイプ分類

| タイプ            | ラベル           | 条件                                              | 例                                 |
| ----------------- | ---------------- | ------------------------------------------------- | ---------------------------------- |
| 💰 投資型成功     | Investment Win   | 金銭リターンあり × 感情的重要度 低〜中            | 義理チョコ→3倍返し                 |
| 💝 感情型成功     | Emotional Win    | 感情的重要度 高（リターン不問）                   | パートナーへの贈り物               |
| 🤝 関係構築型成功 | Relationship Win | 金銭リターンなし × 関係性が前進                   | 上司に渡す→評価面談で好意的        |
| 📈 完全成功       | Perfect Win      | 全層でリターンあり                                | 気になる人→お返しあり+距離が縮まる |
| ⚠️ 要見直し       | Review Needed    | 金銭リターンなし × 感情的重要度 中                | 義理で渡したが反応薄い             |
| ✂️ 損切り推奨     | Cut Loss         | 金銭リターンなし × 感情的重要度 低 × 過去実績なし | 去年もお返しなし。投資の意味なし   |

## 2. スコアリングモデル

### 2.1 入力データ（対象者属性）

**基本属性（既存）:**

| 属性         | 型       | 説明                                        |
| ------------ | -------- | ------------------------------------------- |
| name         | string   | 対象者名                                    |
| relationship | enum     | 上司/同僚/友人/気になる人/パートナー/その他 |
| preferences  | string[] | 好みタグ                                    |
| memo         | string   | 自由記述                                    |

**スコアリング用追加属性:**

| 属性                     | 型             | 説明                                                   |
| ------------------------ | -------------- | ------------------------------------------------------ |
| gender                   | enum           | 男性/女性/その他/未回答                                |
| ageGroup                 | enum           | 10代/20代/30代/40代/50代+                              |
| gaveLastYear             | boolean        | 去年渡したか                                           |
| receivedReturn           | boolean        | ホワイトデーお返しがあったか                           |
| returnValue              | number \| null | お返しの推定金額（円）                                 |
| gaveYearBefore           | boolean        | 一昨年渡したか（2年分の実績）                          |
| receivedReturnYearBefore | boolean        | 一昨年のお返し有無                                     |
| emotionalPriority        | 1-5            | 感情的重要度（1:義理義務 → 5:最重要）                  |
| relationshipGoal         | enum           | 現状維持 / 深めたい / 礼儀として / 距離を置きたい      |
| giriAwareness            | enum           | 義理と認識される / 本命と受け取られる可能性あり / 不明 |

### 2.2 スコアリングロジック

総合スコアは以下の3軸の加重平均で算出する。加重はユーザーの入力（emotionalPriority）によって動的に変動する。

#### 2.2.1 軸① ROIスコア（0〜100）

過去の実績に基づく金銭的リターンの期待値。

| 条件                         | スコア             |
| ---------------------------- | ------------------ |
| 去年お返しあり + 倍率2倍以上 | 90〜100            |
| 去年お返しあり + 倍率1〜2倍  | 70〜89             |
| 去年お返しあり + 等価以下    | 50〜69             |
| 去年渡していない（実績なし） | 50（ニュートラル） |
| 去年お返しなし（1回目）      | 30〜49             |
| 2年連続お返しなし            | 0〜29              |

#### 2.2.2 軸② 関係性スコア（0〜100）

関係性の戦略的価値と方向性。

| 条件                                                               | スコア  |
| ------------------------------------------------------------------ | ------- |
| relationshipGoal = 深めたい + relationship = 気になる人/パートナー | 90〜100 |
| relationshipGoal = 深めたい + relationship = 上司                  | 80〜95  |
| relationshipGoal = 現状維持 + relationship = 上司/同僚             | 60〜79  |
| relationshipGoal = 礼儀として                                      | 30〜59  |
| relationshipGoal = 距離を置きたい                                  | 0〜29   |

#### 2.2.3 軸③ 感情スコア（0〜100）

ユーザーの主観的な気持ち（emotionalPriority をそのまま変換）。

| emotionalPriority | スコア | 意味                        |
| ----------------- | ------ | --------------------------- |
| 5                 | 100    | 最重要。ROI度外視で渡したい |
| 4                 | 80     | かなり大事。多少の損は許容  |
| 3                 | 60     | 普通。バランス重視          |
| 2                 | 40     | 義理寄り。コスパ重視        |
| 1                 | 20     | 完全義理。最低限でいい      |

#### 2.2.4 総合スコアの算出

```
emotionalPriority が高いほど、感情スコアの重みが増し、ROIスコアの重みが減る。

emotionalPriority 1-2（義理寄り）:
  総合スコア = ROI × 0.5 + 関係性 × 0.3 + 感情 × 0.2

emotionalPriority 3（バランス）:
  総合スコア = ROI × 0.3 + 関係性 × 0.4 + 感情 × 0.3

emotionalPriority 4-5（気持ち重視）:
  総合スコア = ROI × 0.1 + 関係性 × 0.3 + 感情 × 0.6
```

### 2.3 ランク判定

| ランク | 総合スコア | 説明                           |
| ------ | ---------- | ------------------------------ |
| S      | 80〜100    | 最優先投資対象。予算を厚く配分 |
| A      | 60〜79     | 重要対象。しっかり投資         |
| B      | 40〜59     | 標準対応。適度な予算で         |
| C      | 0〜39      | 最小限 or 見送り検討           |

### 2.4 成功タイプの自動判定ルール

LLMが以下のルールに基づき、各対象者に成功タイプラベルを付与する。

```
IF emotionalPriority >= 4:
  → 💝 感情型成功（ROI不問で成功とみなす）

ELSE IF 過去にお返し実績あり AND 回収倍率 >= 1.0:
  → 💰 投資型成功

ELSE IF relationshipGoal == "深めたい" AND 関係性スコア >= 70:
  → 🤝 関係構築型成功

ELSE IF emotionalPriority >= 4 AND 過去お返し実績あり:
  → 📈 完全成功

ELSE IF emotionalPriority == 3 AND 過去お返しなし:
  → ⚠️ 要見直し

ELSE IF emotionalPriority <= 2 AND 過去お返しなし:
  → ✂️ 損切り推奨
```

## 3. 予算配分への反映

### 3.1 ランク別配分比率（デフォルト）

| ランク | 配分比率 | 例: 総予算5,000円 |
| ------ | -------- | ----------------- |
| S      | 40%      | 2,000円           |
| A      | 30%      | 1,500円           |
| B      | 20%      | 1,000円           |
| C      | 10%      | 500円             |

※ 同ランク内に複数人がいる場合は、ランク内の予算を均等配分。

### 3.2 感情スコアによる補正

emotionalPriority = 5 の対象者がCランクに入った場合でも、予算の下限を設ける。

```
IF emotionalPriority >= 4 AND ランク == C:
  → 最低予算 = 総予算 × 5% を保証（「気持ちだけは込めましょう」）

IF emotionalPriority == 5:
  → ランクに関わらず、予算配分を1段階上のランク扱いに引き上げ
```

## 4. セールス応用対比

| バレンタイン概念                   | セールス概念                            | 共通フレームワーク         |
| ---------------------------------- | --------------------------------------- | -------------------------- |
| 金銭的リターン（ホワイトデー回収） | 受注・売上・アップセル                  | ROI予測                    |
| 関係性リターン（信頼構築）         | 紹介獲得・長期パートナーシップ          | リレーションシップバリュー |
| 感情的重要度（気持ち重視）         | 戦略的アカウント（赤字でも維持）        | ポートフォリオ戦略         |
| 損切り推奨                         | ロスカット判定                          | リソース最適化             |
| 2年連続お返しなし → 撤退           | 2Q連続未反応リード → ナーチャリング停止 | 離脱判定ルール             |
| 感情スコアによる予算補正           | 戦略アカウントへの赤字投資許容          | 定性判断の定量化           |
